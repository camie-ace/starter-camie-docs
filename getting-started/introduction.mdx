# Introduction to Our Documentation System

Welcome to our modern documentation system built with **MDX**, **ShadCN**, and **React**!

<Alert type="info" title="What is MDX?">
  MDX allows you to write JSX directly in your markdown files, enabling
  interactive documentation.
</Alert>

## Getting Started

<Steps>
  <Step>
    **Install Dependencies** - Set up your project with the required packages
    
    <CodeBlock language="bash">
npm install @mdx-js/mdx @mdx-js/react
    </CodeBlock>
  </Step>
  
  <Step>
    **Configure MDX** - Set up your build system to process MDX files
    
    <CodeBlock language="javascript" title="vite.config.js">
```js
import mdx from '@mdx-js/rollup'

export default {
  plugins: [mdx()]
}
```
    </CodeBlock>
  </Step>
  
  <Step>
    **Create Content** - Start writing your documentation with MDX
    
    <Alert type="success">
    You're ready to start building amazing documentation!
    </Alert>
  </Step>
</Steps>
